# Capabilities

An attempt to create a capability libarary for Rust

## Useful commands

To expand a test

```sh
> cargo expand --test svc
```

```sh
> cargo install cargo-expand
> cargo install cargo-watch
> cargo watch -q -c -x "expand --test svc"
```

svc is the test of code block you wanna run
